# 🏠 국토교통부 실거래가 조회 시스템

국토교통부 공식 실거래가 API를 활용한 아파트 실거래가 조회 및 관심단지 추적 시스템입니다.

## ✨ 주요 기능

### 🔍 아파트 검색
- **계층적 지역 선택**: 시/도 → 군/구 순서로 직관적인 지역 선택
- **아파트명 검색**: 특정 아파트 단지 검색 가능
- **유연한 기간 설정**: 
  - 기간별 조회: 최근 3개월~24개월 데이터 조회
  - 날짜별 조회: 시작일~종료일 직접 선택
- **실시간 데이터**: 국토교통부 최신 실거래가 데이터 활용

### ❤️ 관심단지 관리
- **관심단지 등록**: 검색한 아파트를 관심단지로 등록
- **대시보드**: 관심단지별 최신 거래 정보 및 가격 동향
- **가격 추적**: 월별 가격 변동률 및 동향 분석
- **알림 기능**: 가격 변동 알림 설정 (향후 구현)

### 📊 데이터 분석
- **가격 동향 차트**: Chart.js 기반 인터랙티브 차트
- **통계 정보**: 평균가격, 최고가, 최저가, 거래건수
- **CSV 내보내기**: 검색 결과 및 거래 내역 다운로드
- **상세 분석**: 아파트별 상세 거래 내역 및 분석

## 🛠 기술 스택

- **Backend**: Python, Flask
- **Database**: SQLite
- **Frontend**: HTML, CSS, JavaScript, Bootstrap 5
- **Charts**: Chart.js
- **API**: 국토교통부 공공데이터 API
- **Icons**: Font Awesome

## 📋 요구사항

- Python 3.8+
- 국토교통부 공공데이터 API 키

## ⚙️ 설치 및 실행

### 1. 저장소 클론
```bash
git clone <repository-url>
cd naver_real_estate
```

### 2. 의존성 설치
```bash
pip install -r requirements.txt
```

### 3. API 키 설정 (필수!)
```bash
# .env 파일 생성 및 설정
cp env.example .env
# .env 파일에서 MOLIT_API_KEY=발급받은_키 입력
```

**📖 자세한 API 설정 방법:**
1. [공공데이터포털](https://www.data.go.kr/) 회원가입
2. '아파트매매 실거래 상세자료' API 신청
3. 승인 후 인증키 복사
4. `.env` 파일에 `MOLIT_API_KEY=발급받은_키` 설정

### 4. 프로그램 실행
```bash
python main.py
```

### 5. 웹 브라우저 접속
```
http://localhost:8080
```

## 🔧 환경 설정

`.env` 파일에서 다음 설정을 조정할 수 있습니다:

```bash
# 필수 설정
MOLIT_API_KEY=발급받은_실제_인증키

# 웹 서버 설정
FLASK_HOST=0.0.0.0
FLASK_PORT=8080
FLASK_DEBUG=True
FLASK_SECRET_KEY=your-secret-key-here

# API 호출 설정
API_REQUEST_DELAY=0.1     # API 호출 간격 (초)
API_TIMEOUT=15            # API 타임아웃 (초)
API_MAX_RETRIES=3         # 최대 재시도 횟수

# 로깅 설정
LOG_LEVEL=INFO
```

## 🚀 사용 방법

### 1. 아파트 검색
1. "아파트 검색" 메뉴 클릭
2. 시/도 선택 (예: 서울특별시)
3. 군/구 선택 (예: 강남구)
4. 아파트명 입력 (선택사항)
5. 기간 설정:
   - **기간별 조회**: 최근 3개월~24개월 선택
   - **날짜별 조회**: 시작일과 종료일 직접 선택
6. "검색" 버튼 클릭

### 2. 관심단지 등록
1. 검색 결과에서 하트 아이콘 클릭
2. 또는 아파트 상세페이지에서 "관심단지 추가" 클릭

### 3. 대시보드 확인
1. "대시보드" 메뉴에서 관심단지 현황 확인
2. 가격 동향 및 최신 거래 정보 확인
3. "새로고침" 버튼으로 최신 데이터 업데이트

### 4. 상세 분석
1. 아파트명 클릭하여 상세 페이지 이동
2. 월별 가격 동향 차트 확인
3. 거래 내역 테이블에서 상세 정보 확인
4. CSV 파일로 데이터 내보내기

## 🔧 문제 해결

### API 키 관련 오류
```
ValueError: MOLIT API 서비스키가 필요합니다.
```
**해결방법**: `.env` 파일에 `MOLIT_API_KEY` 설정 확인

### 데이터 없음 오류
```
실거래 데이터를 찾을 수 없습니다.
```
**원인**:
- 해당 지역/기간에 실거래가 없음
- API 키 잘못됨
- 지역코드 매핑 오류

### 네트워크 오류
```
실거래가 조회 중 오류가 발생했습니다.
```
**해결방법**:
- 인터넷 연결 확인
- 방화벽 설정 확인
- 잠시 후 재시도
